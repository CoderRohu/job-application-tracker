# Job Application Tracker

### Developed by Rohan Gouda

Job Application Tracker is a modern web application that helps job seekers manage and track their job applications, interview progress and application status through a clean dashboard interface.

---

## Project Overview

This project was built to solve a real problem faced by students and job seekers — managing multiple job applications efficiently. The system allows users to track applications, update status and organize job search activities.

---

## Features

• Add job applications
• Track application status (Applied / Interview / Rejected / Offer)
• Dashboard statistics
• Search and filter jobs
• Notes section
• Responsive UI design
• Clean dashboard layout
